# Sanity-Anleitung -- Salsa Paderborn

Hallo Nadia! Diese Anleitung zeigt dir Schritt fuer Schritt, wie du Inhalte auf salsa-paderborn.de pflegst. Du brauchst dafuer kein technisches Wissen. Versprochen.

---

## 1. So kommst du ins Studio

Das "Studio" ist dein Bearbeitungsbereich. Dort erstellst und aenderst du alle Inhalte.

1. Oeffne im Browser: **salsa-paderborn.de/studio**
2. Melde dich mit deinem Google-Konto an (das, mit dem du eingeladen wurdest).
3. Fertig. Du siehst jetzt links eine Liste mit den Bereichen: Event, Kurs, Lehrer/in, Tanzschule, Blog-Beitrag.

**Tipp:** Setz dir ein Lesezeichen fuer die Studio-URL, dann findest du es sofort wieder.

---

## 2. Ein Event erstellen

1. Klick links auf **"Event"**.
2. Klick oben rechts auf das **Stift-Symbol** (oder den Button "Erstellen").
3. Fuell die Felder aus:

| Feld | Was eintragen | Beispiel |
|------|--------------|---------|
| **Titel** | Name des Events | "Salsa Night im Kulturwerk" |
| **Slug** | Klick auf "Generate" -- wird automatisch erzeugt | salsa-night-im-kulturwerk |
| **Datum & Uhrzeit** | Wann geht's los? Klick auf den Kalender | 15.03.2026, 21:00 |
| **Enddatum** | Nur bei mehrtaegigen Events (Festivals etc.) | Kann leer bleiben |
| **Kategorie** | Waehle: Party, Social, Festival oder Sonstiges | Party |
| **Location** | Name des Ortes | "Kulturwerk Paderborn" |
| **Stadt** | Die Stadt | "Paderborn" |
| **Kurzbeschreibung** | 1-2 Saetze. Was erwartet die Leute? | "DJ Miguel legt die besten Salsa-Hits auf. Eintritt 8 Euro, Beginn 21 Uhr." |
| **Details** | Laengerer Text, wenn du mehr erzaehlen willst | Kann leer bleiben |
| **Preis** | Was kostet es? | "8 Euro" oder "frei" |
| **Bild** | Ein schoenes Event-Foto (siehe Abschnitt 5) | -- |
| **Externer Link** | Link zur Facebook-Veranstaltung o.ae. | Kann leer bleiben |
| **Veranstalter** | Waehle eine Tanzschule aus der Liste | "Salsa Pasion e.V." |

4. Klick oben rechts auf **"Publish"** (gruen).
5. Geschafft!

---

## 3. Einen Kurs erstellen

1. Klick links auf **"Kurs"**.
2. Klick auf **Erstellen**.
3. Fuell die Felder aus:

| Feld | Was eintragen | Beispiel |
|------|--------------|---------|
| **Titel** | Name des Kurses | "Salsa fuer Anfaenger" |
| **Slug** | Klick auf "Generate" | salsa-fuer-anfaenger |
| **Kategorie** | Kurs, Workshop oder Bootcamp | Kurs |
| **Level** | Anfaenger, Mittelstufe, Fortgeschritten oder Alle | Anfaenger |
| **Tanzstil** | Salsa, Bachata, Kizomba oder Sonstiges | Salsa |
| **Zeitplan** | Wann findet der Kurs statt? | "Dienstags, 19:00 Uhr" |
| **Startdatum** | Falls der Kurs ein festes Startdatum hat | Kann leer bleiben |
| **Ort** | Wo wird getanzt? | "Tanzraum im Kulturwerk" |
| **Lehrer/in** | Waehle aus der Liste | "Nadia" |
| **Tanzschule** | Waehle aus der Liste | "Salsa Pasion e.V." |
| **Beschreibung** | Was erwartet die Teilnehmer? | "Keine Vorkenntnisse noetig. Komm allein oder zu zweit!" |
| **Tanzpartner erforderlich** | Haekchen setzen, wenn ja | Meistens: nein |
| **Bild** | Kursfoto | -- |

4. Klick auf **"Publish"**.

## 4. Einen Blog-Beitrag schreiben

1. Klick links auf **"Blog-Beitrag"**.
2. Klick auf **Erstellen**.
3. Fuell die Felder aus:

| Feld | Was eintragen | Beispiel |
|------|--------------|---------|
| **Titel** | Ueberschrift des Beitrags | "Rueckblick: Latin Vibes Festival 2026" |
| **Slug** | Klick auf "Generate" | rueckblick-latin-vibes-festival-2026 |
| **Veroeffentlicht am** | Datum der Veroeffentlichung | Heute |
| **Autor** | Waehle jemanden aus der Lehrer-Liste | Kann leer bleiben |
| **Vorschautext** | 1-2 Saetze, die in der Uebersicht erscheinen | "Was fuer ein Wochenende! 200 Taenzer, 3 Floors, und ein Finale, das niemand vergisst." |
| **Inhalt** | Der eigentliche Text. Du kannst hier fett, kursiv und Ueberschriften nutzen. | Schreib einfach los! |
| **Bild** | Ein Titelbild fuer den Beitrag | -- |
| **Tags** | Stichwoerter, z.B. "Event-Bericht", "Szene-News" | Tippe ein Wort und drueck Enter |

4. Klick auf **"Publish"**.

## 5. Bilder hochladen -- Tipps

Wenn du bei einem Event, Kurs oder Beitrag auf das Bild-Feld klickst, kannst du ein Bild von deinem Computer hochladen.

**Was gut funktioniert:**
- Querformat (Landschaft) fuer Events und Beitraege
- Mindestens 1200 Pixel breit -- Handy-Fotos sind meistens gross genug
- JPG oder PNG -- beides geht
- Dateien unter 5 MB -- falls ein Bild zu gross ist, verkleinere es vorher (z.B. mit tinypng.com)

**Hotspot setzen:** Nach dem Hochladen siehst du einen Punkt auf dem Bild. Den kannst du verschieben, um festzulegen, welcher Bereich immer sichtbar bleiben soll (z.B. Gesichter). Einfach den Punkt auf den wichtigsten Teil des Bildes ziehen.

**Keine perfekten Fotos noetig!** Ein authentisches Handy-Foto von einem echten Event ist besser als jedes Stock-Foto.

## 6. Was passiert nach dem Speichern?

Nachdem du auf **"Publish"** geklickt hast, ist der Inhalt in Sanity gespeichert. Die Website wird **automatisch zweimal taeglich aktualisiert** (morgens und abends).

Das heisst:
- Deine Aenderung ist **nicht sofort** auf der Website sichtbar.
- Spaetestens am naechsten Tag ist alles online.
- Bei dringenden Aenderungen: Sag Andy Bescheid -- er kann einen sofortigen Update ausloesen.

## 7. Haeufige Fragen

**"Warum sehe ich meine Aenderung nicht sofort auf der Website?"**
Die Website wird zweimal am Tag automatisch neu gebaut. Dein neuer Inhalt ist spaetestens am naechsten Tag sichtbar. Das ist normal und kein Fehler.

**"Kann ich etwas kaputt machen?"**
Nein. Sanity speichert jede Version. Wenn du etwas aus Versehen loeschst oder falsch aenderst, koennen wir es wiederherstellen. Trau dich ruhig, Dinge auszuprobieren.

**"Muss ich mich um die Technik kuemmern?"**
Nein. Du pflegst nur Inhalte -- Texte, Bilder, Termine. Um alles andere kuemmern sich Andy und das System.

**"Was ist der Slug?"**
Der Slug ist der Teil der Webadresse, der nach salsa-paderborn.de/ kommt. Zum Beispiel wird aus dem Event "Salsa Night im Kulturwerk" die Adresse salsa-paderborn.de/events/salsa-night-im-kulturwerk. Klick einfach auf "Generate", dann wird er automatisch aus dem Titel erstellt.

**"Ich habe ein Event eingetragen, das schon vorbei ist. Muss ich es loeschen?"**
Nein. Vergangene Events werden automatisch nicht mehr prominent angezeigt. Du musst nichts loeschen.

**"Ich finde mich nicht zurecht -- was tun?"**
Schreib Andy eine Nachricht. Er hilft dir weiter.
