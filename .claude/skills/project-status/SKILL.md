# Build-Status

1. **Lies den Fortschritt:** `progress/build-log.md`
2. **Lies die Sitemap:** `client/sitemap.md`
3. **Prüfe den tatsächlichen Code:**
   - Welche Pages existieren in `src/pages/`?
   - Welche Komponenten in `src/components/`?
   - Welche Content Collections in `src/content/`?

4. **Präsentiere Andy eine Status-Übersicht:**

```
## Build-Status

### Fertig
- [x] Seite/Feature – Kurzbeschreibung

### In Arbeit
- [ ] Seite/Feature – Was fehlt noch?

### Offen
- [ ] Seite/Feature – Noch nicht angefangen

### Blockiert
- [ ] Seite/Feature – Wartet auf: [was/wen]

### Nächster Schritt
[Empfehlung, was als nächstes gebaut werden sollte und warum]
```

5. **Aktualisiere `progress/build-log.md`** mit dem aktuellen Stand.
